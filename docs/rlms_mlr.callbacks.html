class rlms_mlr.callbacks.base_callback.Callback[source]

Bases: ABC

Base class for all callbacks. Callbacks are used to hook into the training process and perform actions at various stages of training. Each callback can define its own priority, which determines the order in which they are called when multiple callbacks are used in a CallbackList.
